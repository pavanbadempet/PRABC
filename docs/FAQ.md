# FAQ - PRABC

## General Questions

**Q: What is PRABC?**
A: PRABC (Predicting the Risk of Attaining Breast Cancer) is an AI-powered application that estimates the risk of developing breast cancer using machine learning.

**Q: Is this a medical diagnosis tool?**
A: No, this is a screening and estimation tool only. Always consult qualified healthcare professionals for actual medical diagnosis.

**Q: Who should use this?**
A: This tool is designed for educational purposes and general awareness. Women concerned about breast cancer risk should consult healthcare providers.

## Technical

**Q: What machine learning model is used?**
A: The application uses scikit-learn classifiers trained on breast cancer datasets for risk prediction.

**Q: Can I run this locally?**
A: Yes, follow [INSTALLATION.md](INSTALLATION.md) for setup instructions.

**Q: What platforms are supported?**
A: Web (Flask), Android, Windows, and macOS applications are available.

## Features

**Q: What inputs does the app require?**
A: Medical parameters like age, family history, tumor characteristics, and health metrics.

**Q: How accurate are predictions?**
A: Model accuracy depends on input data quality. This should never replace professional medical opinion.

**Q: Can I export my results?**
A: Results can be saved and viewed for personal records or to share with healthcare providers.

## Data & Privacy

**Q: Is my data private?**
A: Yes, no data is shared externally. All processing is local to your device or server.

**Q: Can I delete my data?**
A: Yes, all locally stored data can be deleted anytime.
